Definicion de Flex Box

El modulo de diseño de cja flexible,mas conmunmente conocido como flexbox,es una herramienta poderosa para crear
diseños con CSS.No es tan poderoso como CSS Grid,que veremos en el Capitulo 12,pero puede resolver muchos problemas
de diseño.FlexBox es un diseño unidimensional que puede colocar elementos horizontal o verticalmente(pero no ambos).
Un elemento que usa flexbox como diseño se conoce como contenedor flexible,y los elementos dentro de el son elementos
flexible.

Attardi J. (2020) Flexbox. In: Modern CSS. Apress, Berkeley, CA. https://doi.org/10.1007/978-1-4842-6294-8_10
